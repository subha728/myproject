# myproject
Personal Developer Portfolio – Full Stack Showcase

##Project Overview
A responsive full-stack personal portfolio website designed to showcase projects, skills, and contact details. It includes a dynamic project section where data is managed using a backend API and stored in a database. This project demonstrates integration of frontend, backend, and database in a real-world deployment setup.

##Features
- Responsive and modern UI design  
- Project showcase section (dynamic content from backend)  
- About and Skills section  
- Contact form with backend integration  
- REST API for handling project data  
- Full-stack architecture

  
##Tech stack

###Frontend
  HTML5
  CSS3  
  JavaScript
  
###Backend
- Node.js  
- Express.js
  
###Database
- MongoDB (MongoDB Atlas)
